Pree

Record spoken word without the need for editing. Hit one button when you begin speaking. Hit another to stop, then choose whether to keep or discard your clip. When you are done, you will have a single WAV file with only the audio you want.

If you are recording video as well, sync the second WAV file to the video channel. Visible markers within the sound track show you exactly where to cut. Line up the audio in Camtasia, and process the *.camproj with Pree to automatically edit your video.

Don’t fix it in post. Capture in Pree.


The following prerequisites are required:


  • Microsoft .NET Framework 4.5.1 (x86 and x64)

Click the link below to install the prerequisites and run the application.

Install


With a presenter remote (good for standing at mic)

  • Next (PgDn) - Record
  • Prev (PgUp) - Stop, toggle between keep and discard

While another app has focus (good for demos)

  • Ctrl+Shift+F10 - Record
  • Ctrl+Shift+F11 - Stop, toggle between keep and discard

The recording automatically trims 200 ms from the beginning and end of each clip to mask the sound of the keyboard or remote. Pause for half a beat before speaking. Practice before using the tool in anger.

Intro to Pree from Michael L Perry on Vimeo.

The tool outputs three files:

*.wav The edited wav file.
*_time.wav The timeline wav file, with solid tone where the video should be cut.
*.log Time codes for the segments that should be kept.


Record with Camtasia simultaneously (make sure that you record sound). Drop the *_time.wav and *.camrec files into a Camtasia project and line up the sound. Save and close the project, then open the *.camproj file in Pree. It will automatically edit the video for you! The result will be saved to *_trimmed.camproj.
